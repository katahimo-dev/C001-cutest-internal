## 前文

甲および乙は、「保育業務管理システム Firebase統合開発」（以下「本プロジェクト」という）に関連して、**相互に開示される秘密情報の取扱い**について、以下のとおり合意する。

本契約は相互義務を規定するものとし、甲・乙のいずれかが開示した秘密情報についても、開示側と受領側の双方が以下の義務を負う。

---

## 第1条（秘密情報の定義）

### 1.1 秘密情報

本契約において「秘密情報」とは、甲または乙が相手方に開示する以下の情報をいう。形式（書面、口頭、電子データ、その他一切の形式）を問わない。

1. **事業情報**: 事業計画、戦略、財務情報、顧客リスト、取引条件、価格情報
2. **技術情報**: システム仕様、設計書、ソースコード、アルゴリズム、ノウハウ
3. **顧客情報**: 保育施設利用者、スタッフ、関係者の個人情報
4. **業務情報**: 業務フロー、運用ルール、社内規程、帳票フォーマット
5. **その他**: 秘密であることを明示した情報

### 1.2 除外情報

以下のいずれかに該当する情報は、秘密情報に含まれない。

1. 開示時点で既に公知であった情報
2. 開示後、受領者の責によらず公知となった情報
3. 開示時点で受領者が既に保有していた情報（証明可能な場合に限る）
4. 正当な権限を有する第三者から秘密保持義務を負うことなく取得した情報
5. 受領者が秘密情報を使用することなく独自に開発した情報

---

## 第2条（秘密保持義務）

### 2.1 基本的義務

甲および乙は、相手方の秘密情報について、以下の義務を負う。

1. **目的外使用の禁止**: 秘密情報を本プロジェクトの遂行以外の目的で使用しない
2. **漏洩防止**: 秘密情報を第三者に開示、漏洩しない
3. **善管注意義務**: 自己の秘密情報と同等以上の注意をもって管理する
4. **複製制限**: 業務上必要な範囲を超えて複製しない
5. **アクセス制限**: 業務上必要最小限の者のみがアクセスできるよう管理する

### 2.2 例外的開示

以下の場合、秘密情報を第三者に開示することができる。ただし、当該第三者に本契約と同等の秘密保持義務を課すものとする。

1. 相手方の書面による事前承諾を得た場合
2. 弁護士、税理士、公認会計士等の法令上の守秘義務を負う専門家に相談する場合
3. 法令または裁判所、行政機関の命令により開示が義務付けられた場合（この場合、事前に相手方に通知し、開示範囲を最小限に留めるよう努力する）

## 第3条（個人情報の特別保護）

### 3.1 個人情報の定義

本契約における「個人情報」とは、秘密情報のうち、個人情報の保護に関する法律（以下「個人情報保護法」という）第2条に定める個人情報をいう。

### 3.2 取り扱う個人情報の範囲

本プロジェクトにおいて取り扱う可能性のある個人情報は、以下のとおりである。

| カテゴリ | 情報項目 | 取扱いレベル |
|----------|----------|--------------| 
| **利用者情報** | 氏名、住所、電話番号、生年月日 | 最高 |
| **児童情報** | 氏名、生年月日、アレルギー情報、健康状態 | 最高 |
| **スタッフ情報** | 氏名、連絡先、勤務情報、資格情報 | 高 |
| **緊急連絡先** | 保護者氏名、電話番号、住所 | 高 |

### 3.3 乙の特別義務

乙は、個人情報について、第2条の義務に加え、以下の特別義務を負う。

1. **安全管理措置**
   - 開発端末の暗号化（BitLocker、FileVault等）の適用
   - 強固なパスワードおよび二要素認証の使用
   - 開発環境への最新セキュリティパッチの適用
   - ウイルス対策ソフトウェアの導入・更新

2. **開発環境での取扱い**
   - 本番データを開発・テスト環境で原則使用しない
   - やむを得ず使用する場合は、匿名化・仮名化処理を行う
   - 開発端末への個人情報の長期保存を避ける

3. **通信の安全確保**
   - 個人情報を含むデータの送受信は暗号化通信（HTTPS、SSH等）を使用
   - 公衆Wi-Fi環境での個人情報へのアクセスを避ける

4. **アクセスログの管理**
   - 本番環境の個人情報へのアクセスは記録を残す
   - 不審なアクセスを検知した場合は甲に報告する

### 3.4 委託先の監督

乙が甲の承諾を得て個人情報の取扱いを第三者に委託する場合、当該委託先に対し、本契約と同等以上の義務を課し、適切に監督する。

## 第4条（情報漏洩時の対応）

### 4.1 漏洩の定義

本契約における「情報漏洩」とは、以下の事態をいう。

1. 秘密情報が権限のない第三者に開示・取得された場合
2. 秘密情報が紛失・盗難された場合
3. 秘密情報が不正にアクセス・改ざん・破壊された場合
4. 上記のおそれがある場合

### 4.2 発生時の報告義務

1. 情報漏洩を発見した場合、**直ちに（発見から24時間以内を目途に）** 相手方に報告する。
2. 報告は、以下の事項を含むものとする（判明している範囲で）。
   - 発生日時・発見日時
   - 漏洩の概要・原因
   - 漏洩した情報の種類・件数
   - 二次被害のおそれ
   - 既に講じた措置・今後の対応方針

### 4.3 初動対応

情報漏洩を発見した場合、以下の初動対応を直ちに行う。

1. 被害拡大防止措置（アクセス遮断、パスワード変更等）
2. 証拠保全（ログの保存、状況の記録）
3. 原因調査の開始
4. 相手方への継続的な状況報告

### 4.4 対応協力

1. 相手方の指示に従い、情報漏洩への対応に全面的に協力する。
2. 個人情報保護委員会への報告、本人への通知等が必要な場合、相手方に協力する。

### 4.5 費用負担

情報漏洩が過失に起因する場合、対応に要した合理的費用は過失側が負担する。ただし、賠償責任保険の適用を受ける場合はこの限りではない。

## 第5条（賠償責任保険）

### 5.1 保険加入義務

1. 乙は、本契約期間中、フリーランス協会の「賠償責任保険」または同等の保険に加入し、これを維持する。
2. 保険は、以下の事故を対象とするものとする。
   - 情報漏洩に起因する損害賠償責任
   - 著作権侵害に起因する損害賠償責任
   - 納品物の瑕疵に起因する損害賠償責任
   - その他業務遂行に起因する損害賠償責任

### 5.2 保険の内容

| 項目 | 内容 |
|------|------|
| **保険名** | フリーランス協会 賠償責任保険（または同等） |
| **補償限度額** | 1事故あたり1,000万円以上 |
| **対象期間** | 本契約期間と同期間 |

### 5.3 証明書の提示

乙は、甲の要請があった場合、保険加入を証明する書類を5営業日以内に提示する。

### 5.4 保険の失効

保険が何らかの理由で失効した場合、乙は速やかに甲に通知し、代替保険への加入等の措置を講じる。

## 第6条（秘密情報の返還・削除）

### 6.1 契約終了時の処理

本契約または本プロジェクトに関する準委任契約が終了した場合、相手方の指示に従い、以下のいずれかを行う。

1. **返還**: 秘密情報（複製物を含む）を相手方に返還する
2. **削除**: 秘密情報（複製物を含む）を復元不能な方法で削除する

### 6.2 削除の方法

電子データの削除は、以下のいずれかの方法による。

1. 専用ソフトウェアによる完全消去（複数回上書き）
2. 記録媒体の物理的破壊
3. 暗号化されたデータの暗号鍵の破棄

### 6.3 削除証明書

削除完了後、以下の事項を記載した削除証明書を相手方に提出する。

- 削除した情報の概要
- 削除日
- 削除方法
- 削除実施者

### 6.4 例外

以下の情報は、合理的な期間保持することができる。

1. 法令により保存が義務付けられている情報
2. バックアップシステムに自動保存された情報（合理的期間経過後に自動削除される場合）
3. 本契約の履行を証明するために必要な最小限の情報

## 第7条（監査）

### 7.1 監査権

相手方の秘密情報管理状況を確認するため、合理的な事前通知（原則5営業日前）の上、事業所を訪問し、または書面で報告を求めることができる。

### 7.2 協力義務

監査に誠実に協力し、合理的な範囲で情報を提供する。

### 7.3 監査費用

監査に要する費用は、原則として監査を実施する側が負担する。ただし、監査の結果、相手方の重大な義務違反が発覚した場合は相手方が負担する。

## 第8条（有効期間）

### 8.1 契約期間

本契約の有効期間は、契約締結日から本プロジェクトに関する準委任契約が終了する日までとする。

### 8.2 秘密保持義務の存続

本契約終了後も、秘密情報に関する秘密保持義務は、以下の期間存続する。

| 情報の種類 | 存続期間 |
|------------|----------|
| 個人情報 | 永久（法令上の保存期間経過後は削除義務） |
| 技術情報 | 契約終了後5年間 |
| 事業情報 | 契約終了後5年間 |
| その他の秘密情報 | 契約終了後3年間 |

## 第9条（損害賠償）

### 9.1 賠償責任

本契約に違反し、相手方に損害を与えた場合、直接かつ現実に生じた損害を賠償する。

### 9.2 賠償額の上限

賠償責任の上限は、以下のとおりとする。

1. **通常の秘密情報の漏洩**: 準委任契約における過去12ヶ月間の報酬総額
2. **個人情報の漏洩**: 賠償責任保険の補償限度額（1,000万円以上）
3. **故意または重過失による漏洩**: 上限なし

### 9.3 保険金の優先適用

損害賠償の支払いにあたっては、まず賠償責任保険からの保険金を充当し、不足分を当該当事者が負担する。

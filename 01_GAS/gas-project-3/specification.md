# gas-project-3 詳細仕様書

## 1. 概要
本プロジェクトは、各スタッフが個別に所有する「出勤簿」スプレッドシートに適用される補助スクリプトです。主にユーザーの編集操作に対する視覚的フィードバックと、ファイル起動時の利便性向上を目的としています。

## 2. 主な機能
### 2.1. 編集セルの自動着色 (`onEdit` トリガー)
- **機能**: ユーザーがスプレッドシートのセルを編集した際に、そのセルの背景色を自動的に薄赤色 (`#fce4e4`) に変更します。
- **適用条件**:
    - ファイル名に「出勤簿」という文字列が含まれていること。
    - 編集されたシート名が「1月」〜「12月」のいずれかであること（正規表現 `/^(1[0-2]|[1-9])月$/`）。
    - 編集箇所がデータ開始行（2行目以降）であること。
- **目的**: どの項目が手動で修正・入力されたかを一目で判別できるようにします。

### 2.2. アクティブシートの自動切り替え (`onOpen` トリガー)
- **機能**: スプレッドシートを開いた際、自動的に「現在の月」に対応するシートをアクティブ表示にします。
- **動作**: 実行時の月を取得し（例: 3月なら「3月」）、その名称のシートが存在すれば自動的にタブを切り替えます。
- **目的**: 月末や月初にシートを探す手間を省き、即座に入力作業に移れるようにします。

## 3. 導入方法
本スクリプトは、一括作成される各スタッフの出勤簿テンプレートに組み込まれる、または各個人のファイルにコンテナバインドスクリプトとして配置されます。

## 4. 注意事項
- スプレッドシートの標準トリガー（シンプルトリガー）を使用しているため、ユーザーのブラウザ操作時のみ機能します（プログラムによる書き込みには反応しません）。
- 背景色のリセット機能は含まれていないため、色のクリアは手動または別のメンテナンス用スクリプトで行う必要があります。

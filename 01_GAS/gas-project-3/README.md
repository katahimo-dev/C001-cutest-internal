# gas-project-3

スタッフ個別の出勤簿スプレッドシートに組み込む補助GAS。
シンプルトリガー中心の軽量プロジェクト。

## 目的

- 手動編集セルを視覚的に識別。
- 月次入力シートへの導線を自動化。

## 主な機能

- `onEdit`
  - 条件に合致する編集セルを薄赤色 (`#fce4e4`) へ自動着色。
  - 対象条件
    - ファイル名に「出勤簿」を含む。
    - シート名が `1月` - `12月`。
    - 2行目以降。
- `onOpen`
  - ファイル起動時に当月シートへ自動切替。

## 導入位置

- 出勤簿テンプレートに組み込み。
- または各スタッフのコンテナバインドスクリプトとして配置。

## 注意点

- シンプルトリガーのため、ユーザー操作時のみ発火。
- 背景色のリセット機能は非搭載。
- 詳細仕様は `specification.md` を参照。

# gas-integrated-system

スタッフ本人と管理者が、スマホから出勤簿を直接修正できるGAS Webアプリ。

## 目的

- 出勤簿修正のリアルタイム化。
- 権限制御付きの安全な勤怠入力。
- 管理者向け運用機能（配布、閲覧、一括更新）の提供。

## 主な機能

- `doget.js` + `index.html`
  - 日付指定で個人出勤簿を読込/更新。
  - 入力規則連携によるプルダウン表示。
  - 修正セルの可視化（背景色）。
- 権限管理
  - 一般スタッフ: 自分の出勤簿のみ操作。
  - 管理者: 全スタッフを選択して閲覧/修正。
- 編集期限
  - 当月内データのみ修正許可。
- 運用補助
  - アプリURLの一括送信。
  - `STAFF_SYNC.js` でスタッフ情報同期。
  - カレンダー連携データの直接反映（関連機能）。

## 必須設定

- Staff名簿シート（メールアドレス、管理者フラグ等）。
- スタッフごとの出勤簿ファイル命名規則。
- 年度設定（自動算出/上書き設定）の確認。

## 運用手順

1. スタッフ情報を同期。
2. 管理者でWebアプリ動作を確認。
3. 一般スタッフにURL配布。
4. 月次運用で権限・期限判定を定期点検。
